ngClass and ngStyle are two powerful built-in directives that allow you to dynamically set CSS classes and inline styles on elements based on conditions in your component.
ngClass Directive : ngClass dynamically adds or removes CSS classes to elements based on a condition.
Syntax : [ngClass]="classExpression"
Examples:

1. Adding a single class conditionally:
This div is active
Here above, the active class will be added to the div if isActive is true.
2. Adding multiple classes based on conditions:
This div can be active or disabled
In this case, active will be applied if isActive is true, and disabled will be applied if isDisabled is true.
3. Binding to an array of class names:
This div has multiple classes
ngStyle Directive
ngStyle allows you to set inline styles dynamically on elements.
Syntax: [ngStyle]="styleExpression"
Examples:

1. Applying a single style dynamically:
This text has dynamic color
2. Applying multiple styles dynamically:
This text has dynamic color and font size
3. Conditional styles:
This div has conditional background color
